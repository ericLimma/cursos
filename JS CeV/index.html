2. Mudar a cor de fundo com JavaScript

Adicione um botão à página. Quando o botão for clicado, altere a cor de fundo da página para uma cor aleatória.

3. Contador com incremento e decremento

Crie um contador que começa em 0. Adicione dois botões: um para incrementar o contador e outro para decrementá-lo. Exiba o valor do contador na página.
