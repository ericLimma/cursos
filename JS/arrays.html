Java Script - Arrays

Array unidimensional

Arrays - Estruturas de dados que armazenam uma coleção ordenada de elementos, como números ou strings, permitindo acesso e manipulação através de índices. Exemplo de uso 


        let frutas = ['abacate', 'melancia', 'morango', 'uva', 'maracuja'];
        frutas[1] //'melancia';

        //ou

        let frutas = Array();

        frutas[0] = 'abacate';
        frutas[1] = 'melancia';
        frutas[2] = 'morango';
        frutas[3] = 'uva;
        frutas['frutaAzeda'] = 'maracuja';  //um indice não precisa necessariamente ser um numero.

        frutas.push('laranja'); //'Empurra' para o ultimo indice no array.
        let ultimaFruta = frutas.pop(); // Apaga o utlimo elemento do array e retorna o valor removido.

        frutas.unshift('marmelo'); //adciona um elemento na priemeira posição
        let primeiraFruta = frutas.shift();// Apaga o utlimo elemento do array e retorna o valor removido.

Arrays multidimensionais

Arrays multidimensionais são arrays que contêm outros arrays como seus elementos, formando uma estrutura semelhante a uma tabela, matriz ou cubo. Por exemplo, um array 2D (bidimensional) é uma lista de listas, como uma tabela com linhas e colunas. Eles permitem armazenar dados em mais de uma dimensão, facilitando o trabalho com estruturas complexas. Exemplo:


            let matriz = [
            [1, 2, 3],  // Primeira linha
            [4, 5, 6],  // Segunda linha
            [7, 8, 9]   // Terceira linha
          ];

        let frutas = ['abacate', 'melancia', 'morango', 'uva', 'maracuja'];

        frutas.indexOf('abacate'); //retorna indice 0;
