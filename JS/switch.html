Switch case

Exemplo:

    
        let age = 15;
        let result = document.getElementById('result');

        switch (true) {
            case age < 15:
                result.innerHTML = '👶Criança';
                break;
                
            case age >= 15 && age < 30:
                result.innerHTML = '🧑‍🎓Jovem';
                break;

            case age >= 30 && age < 60:
                result.innerHTML = '👨‍💼Adulto';
                break;

            case age >= 60:
                result.innerHTML = '🧓Idoso';
                break;

            case age >= 150:
                result.innerHTML = '🧟Zumbi';
                break;
                
            default:
                result.innerHTML = 'Idade não categorizada';
        }

Obs.: O Switch apenas compara identicos, caso compare '2'(string) com 2(number), o switch retornará o default
