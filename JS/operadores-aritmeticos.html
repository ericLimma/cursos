Operadores Aritméticos

Adição (+) - Soma Valores, ex.: 

                
                    let num1 = 1;
                    let num2 = 2;
                    num1 + num2 // 3

Subttração (-) - Diferença de Valores 

            
                let num1 = 1;
                let num2 = 2;
                num1 - num2 // 0

Multiplicação (*) - Produto de Valores 

            
                let num1 = 2;
                let num2 = 3;
                num1 * num2 // 6

Divisão (/) - Quociente de Valores 

            
                let num1 = 2;
                let num2 = 4;
                num1 / num2 // 0.5

Módulo (%) - Resto existente em uma operação de divisão. 

            
                let num1 = 5;
                let num2 = 2;
                num1 % num2 // 1 (4 / 2 = 0)

Incremento (++) - Ex.: 


                let a = 1;
                a++ = 2;

Decremento (--) - Ex.: 


                let a = 1;
                a-- = 0;


Operações aritméticas na atribuição de valores:

exemplo:


            let test = 10;
            teste += 2; // 12
            teste -= 2; // 8
            teste *= 2; // 20
            teste /= 2; // 5
            teste %= 2; // 0
