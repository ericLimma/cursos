Funções Nativas de Manipulação de strings.


          let str = "Aumari";
          str.lenght; // 6 - conta os caracteres
          str.charAt(3); // A - retorna o caracter no indice passado como parametro
          str.indexOf('a'); // 0 - retorna o indice da primeira ocorrência do caracter passado como parametro
          str.replace(/a/g, '@'); // pesquisa o primeiro parâmetro e substitui pelo segundo,
                                  o /g faz com que todas as ocorrencias seja substituidas.
          escreva(str.toLowerCase()); // transforma caracteres em minusculo
          escreva(str.toUpperCase()); // transforma caracteres em maiusculo

Funções Nativas de Data

Útil para pesquisas e marcações temporais.


          let date = new Date; // instancia um objeto do tipo data.
          date.getDate(); // fornece o dia do mês
          date.getDay(); // fornece o dia da semana.
          date.getMonth() + 1; // fornece o mês.
          date.getHours(); // fornece a hora.
          date.getMinutes(); // fornece os minutos.
          date.getSeconds(); // fornece os segundos.
          date.getMilliseconds(); // fornece os Millisegundos
          date.getFullYear(); //fornece o ano atual

          //set date

          date.setDate(date.getDate() + 720) // soma 720 dias ao dia atual.
          date.setMonth(date.getMonth() + 5) // soma 5 meses a partir do mes atual.
          date.setFullYear(date.getFullYear() + 5) //soma 5 anos a partir do ano atual.
