Java Script - Eventos

Eventos de Mouse

onclick( )
ondblclick( )
onmouseup( )
onmouseover( )
onmouseout( )

*Mais de um evento pode ser colocado no mesmo elemento
**nem todos os eventos são suportados por todos os elementos

Eventos de Teclado

onkeydown - quuando pressiona uma tecla

onkeyup - quando solta uma tecla

onkeypress - quando pressiona uma tecla que representa um caracter

Janela - Window

onresize - quando a aba do navegador é redimensionado
*Exclusivo do elemento body

onscroll - quando ha a rolagem da barra

Eventos de Formulário

onfocus - quando clicamos nele para iniciar a inserção de caracteres

onblur - quando clicamos em outro elemento

onchange - quando mudamos algo no input
